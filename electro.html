Внутрішні та зовнішні електромонтажні роботи

  • монтаж електрощитового обладнання для будинків, цехів, офсів
  • монтаж і налагодження обладання трансформаторних підстанцій різних типів( ТП, КТП, ЗТП )
  • проведення реконструкції існуючої мережі з використанням сучасних електротехнічних матеріалів що зменшують витрати на обслуговування і підвищують надійність
  • аварійний ремонт електромережі споживачів
  • монтаж та налагодження конденсаторних установок ( ККУ ) для зменшення витрат електроенергії
  • монтаж і налагодження автоматики резервного електропостачання ( АВР, ВРУ )
  • монтаж електрощитів ( ВРП, ГРЩ, РЩ, ЩС, ЩО )
  • заміну та ремонт електрообладнання в існуючих електричних щитах
  • проведення робіт по узгодженню навантаження електрообладнання з дозволеною потужністю
  • монтаж зовнішніх, фасадних та внутрішніх мереж освітлення об′єктів
  • улаштування систем обліку електроенегрії прямого та трансформаторного увімкнення
